This section will discuss C and C++.

The C programming language was created by Dennis Ritchie in the 1970's. After its development, C would soon spread all over the world and eventually becoming the worlds most used programming language.

There were many different languages derived from the C language. C#, C--, and the most popular of the three, C++.

C++ is a widely used and popular coding language. While it may not be as popular as its parent language, it is commonly taught as an introdution to other languages.

The C++ language was created with the thought of adding to the already great C language. This was done through the adding of many different classes.
